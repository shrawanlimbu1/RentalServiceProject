# RentalServiceProject

A full-stack bike rental service application built with React and Node.js.

## Features

- User authentication and authorization
- Bike catalog browsing
- Rental management
- Admin dashboard
- User dashboard

## Tech Stack

**Frontend:**
- React
- React Router
- Context API for state management

**Backend:**
- Node.js
- Express.js
- MySQL database

## Getting Started

### Prerequisites
- Node.js
- MySQL

### Installation

1. Clone the repository
```bash
git clone https://github.com/shrawanlimbu1/RentalServiceProject.git
cd RentalServiceProject
```

2. Install backend dependencies
```bash
cd backend
npm install
```

3. Install frontend dependencies
```bash
cd ../frontend
npm install
```

4. Set up your database and environment variables

5. Run the application
```bash
# Backend (from backend directory)
npm start

# Frontend (from frontend directory)
npm run dev
```
